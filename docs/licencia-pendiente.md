# Licencia pendiente

La licencia definitiva del proyecto queda pendiente de confirmacion por Joel Concepcion Villanueva.

## Recomendacion inicial

- **Codigo:** MIT, si se quiere facilitar reutilizacion tecnica del HTML, CSS, JavaScript, scripts y esquemas.
- **Contenidos pedagogicos:** CC BY 4.0 si se permite reutilizacion amplia con atribucion, o CC BY-NC 4.0 si se prefiere limitar usos comerciales.

## Criterio recomendado

Separar licencia de codigo y licencia de contenidos evita confundir:

- estructura tecnica reutilizable;
- textos, fichas, criterios pedagogicos, documentacion y marca personal;
- fuentes oficiales citadas, que no pertenecen al proyecto.

## Advertencia

Hasta que se confirme la licencia, el README debe mantener la licencia como pendiente. No conviene crear archivos `LICENSE` definitivos ni afirmar permisos de reutilizacion cerrados sin decision expresa del titular.

## Texto sugerido para una decision futura

Cuando se confirme, se podrian crear:

- `LICENSE-CODE.md` para codigo;
- `LICENSE-CONTENT.md` para contenidos;
- una seccion breve en README explicando alcance y atribucion.
